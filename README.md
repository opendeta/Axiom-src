# 🌐 Axiom — Mobile Automation Browser

Axiom is a powerful, open-source Android web browser designed specifically for mobile automation. It allows you to run tasks, scrape data, and manage custom bots efficiently right from your smartphone.

---

## ✨ Key Features

* **🤖 Immortal Bot:** Run automation tasks on multiple websites simultaneously without performance drops.
* **⚡ Native JS Bridge:** Enhanced communication between the Android system and web pages.
* **🔓 Open Source:** Completely transparent, ad-free, and secure with the GPL-3.0 License.

## 🛠️ Custom Bot Interfaces

Axiom comes with custom JavaScript native bridges to give you deep control over the browser session:

* `Code.Bot()` — Instantly view the raw page source code directly within your automation script.
* `Cleaner.Bot()` — Programmatically clear site cookies and session data to maintain fresh instances.
* `Bot.search()` — Native search execution bridge optimized for high-speed bot queries.
* `AppBot()` — Triggers Android system-level actions including clicks, swipes, and scrolls.
